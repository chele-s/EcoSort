```mermaid

    A[EcoSortSystem] --> B[ConfigManager]
    A --> C[ComponentManager]
```

### Flujo de Procesamiento

1. **Detecci√≥n**: Sensor de trigger detecta objeto
2. **Captura**: C√°mara captura imagen del objeto
3. **Clasificaci√≥n**: IA analiza y clasifica el material
4. **C√°lculo**: Sistema calcula timing para activaci√≥n
5. **Desviaci√≥n**: Actuador dirige objeto a tolva correcta
6. **Registro**: Base de datos almacena resultado
7. **Monitoreo**: Sensores verifican niveles de tolvas
